    Overview

    Installation

    Can be managed using yarn, npm, or jspm.

    yarn

    yarn add @graph-data-structure/specification

    npm

    npm install @graph-data-structure/specification --save

    jspm

    jspm install npm:@graph-data-structure/specification

    Usage

    :warning: Depending on your environment, the code may require regeneratorRuntime to be defined, for instance by importing regenerator-runtime/runtime.

    First, require the polyfill at the entry point of your application

    await import('regenerator-runtime/runtime.js');
// or
import 'regenerator-runtime/runtime.js';

    Then, import the library where needed

    const spec = await import('@graph-data-structure/specification');
// or
import * as spec from '@graph-data-structure/specification';
